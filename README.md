# hbase-collectd
Collectd plugin for HBase

### Installation

1. Place hbase_collector.py ing /var/lib/collectd/ and hbase.conf in /etc/collectd.d/
2. Change hbase_collector variables to fit your config
3. Restart collectd and watch!
